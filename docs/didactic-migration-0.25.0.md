# Schriftliche Verfahren in Version 0.25.0

Stand: App 0.25.0, Katalog 0.24.0, Schema 18. Die Umsetzung ist intern konsistent geprüft, aber nicht durch eine Lehrkraft evaluiert.

## Fachliche Änderung

Schriftliche Addition und Subtraktion werden als verdichtete Stellenwertverfahren aufgebaut. Das Kind richtet zunächst beide Zahlen aus, bearbeitet die Spalten konsequent von rechts nach links und versteht einen Übertrag beziehungsweise eine Entbündelung als Bündelungshandlung. Eine Umkehraufgabe bildet den Transfer.

| Phase | Schriftliche Addition | Schriftliche Subtraktion |
| --- | --- | --- |
| `activate` | zweiten Summanden stellengerecht eintragen | Subtrahenden stellengerecht eintragen |
| `understand` | zehn Einer als einen neuen Zehner deuten | eine größere Einheit in zehn kleinere Einheiten tauschen |
| `guided-practice` | ohne Übertrag spaltenweise rechnen | ohne Entbündelung spaltenweise rechnen |
| `independent-practice` | einen sichtbaren Einerübertrag notieren und mitrechnen | eine sichtbare Entbündelung ausführen |
| `automate` | genau einen Einer- oder Zehnerübertrag selbstständig erkennen | genau eine Entbündelung selbstständig erkennen |
| `transfer` | Summe mit einer Subtraktion prüfen | Differenz mit einer Addition prüfen |

## Darstellung und Rückmeldung

Die zweite Zahlenzeile bleibt während der Ausrichtungsaufgabe maskiert. Ergebnisziffern werden ausschließlich in der tatsächlich bearbeiteten Reihenfolge Einer, Zehner, Hunderter aufgedeckt. Übertrag und Entbündelung erscheinen nur, wenn sie als Lernhandlung bearbeitet wurden oder nach einem Fehler als Hilfe nötig sind. Fehlvorstellungskennungen führen zu konkreten Rückmeldungen über Stellenwert, Tauschrichtung und Probe.

## Abgrenzung

Produktiv sind Aufgaben ohne Übergang sowie genau ein Übertrag oder genau eine Entbündelung. Mehrere Übergänge, Entbündelung über Nullstellen, schriftliche Multiplikation und schriftliche Division bleiben ausgeschlossen. Die Eingabe der auszurichtenden Zahl prüft die Stellenfolge, ersetzt aber noch keine frei verschiebbare Ziffernlegung.
